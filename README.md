## Local Development

### Environment

- [Node.js](https://nodejs.org): 18.0+.
- [Pnpm](https://pnpm.io): 3.0+.

### Installation

```sh
pnpm install
```

### Development

```sh
pnpm start
```

### Build

```sh
pnpm build
```
